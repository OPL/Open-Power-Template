Title: setCache()
ShortTitle: Opt_View::setCache()
Reference: void Opt_View([Opt_Caching_Interface $cache = null])
VersionSince: 2.0-RC1
SeeAlso:
 - api.opt-view.get-cache
 - guide.cache
 
----

Registers a new view caching interface, overwriting the global caching settings. The method called without parameters, removes the existing caching interface from the view.
