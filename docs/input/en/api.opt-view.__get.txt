Title: __get()
ShortTitle: Opt_Class::__get()
Reference: mixed __get(string $name)
VersionSince: 2.0-beta2
Status: public
SeeAlso:
 - api.opt-view.get

----

This magic method provides an alternative way to read the local template variable value:

~~~~
[php]
echo $view->variable;
~~~~

> [help]
> It is recommended to use the magic method instead of [api.opt-view.get] unless we are not going to specify a dynamic variable name.
