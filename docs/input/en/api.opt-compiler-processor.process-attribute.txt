Title: processAttribute()
ShortTitle: Opt_Compiler_Processor::processAttribute()
Reference: void processAttribute(Opt_Xml_Node $node, Opt_Xml_Attribute $attribute)
Status: extendable public
SeeAlso:
 - api.opt-compiler-processor.postprocess-attribute
 - api.opt-compiler-processor.process-node

---

This method captures all the XML attributes registered by the processor and allows to parse them much like [api.opt-compiler-processor.process-node] does with XML tags. The taken arguments are the registered `$attribute` and the XML `$node` which the attribute is assigned to.
